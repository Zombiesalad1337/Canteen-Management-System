# Canteen-Management-System

Command line software to facilitate item management, ordering and bill creation.
Written in C++ and uses the SQLite3 database.

Running the code:
1. Install sqlite3 on your system.
2. Clone this github repository.
3. Compile main.cpp : ```$ g++ main.cpp -l sqlite3 -o main.out```
4. Run main.out : ```$ ./main.out```
